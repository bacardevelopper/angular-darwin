Je suis developpeur web (fullstack). ça veut dire que je suis capable de conçevoir un site web de A à Z ou creer des outils particuliers pour vous. Mes compétences clés (skills)

Javascript : Langage de programmation qui permet de programmer un applicaion web. Flexible grace à l'environnement de frameworks autours du langage

Angular : Framework qui permet de creer le plus rapidement un site internet coté frontend (graphique) rapidement.

NodeJs : runtime (environnement d'execution) qui permet de creer un serveur en javascript (backend), ou des outils plus poussés. Avec les frameworks et cli quelle integre. Elle permet de developper rapidement des outils liées à differents domaines comme le developpement d'un bacend en Javascript.

Git : Logiciel (cli) est un logiciel de gestion de versions décentralisé. C'est un logiciel libre créé par Linus Torvalds. Elle permet de gerer la version de notre code : (on peut enregistrer, modifier, supprimer, ajouter, travailler en equipe sur le meme code source).

MongoDb : Systeme de gestion base de donnée (noSql) : permet de une base de donnée avec une strucure moins strict, le format de donnée enregistrer est le JSON.

NPM : Logiciel (cli) : npm est le gestionnaire de paquets officiel de Node.js. Depuis la version 0.6.3 de Node.js, npm fait partie de l'environnement et est donc automatiquement installé par défaut. npm fonctionne avec un terminal et gère les dépendances pour une application
